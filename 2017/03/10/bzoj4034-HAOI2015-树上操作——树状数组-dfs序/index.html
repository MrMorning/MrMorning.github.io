Posted on 2017-03-10 Edited on 2020-10-23

Brief Description

您需要设计一种数据结构支持以下操作：
1. 把某个节点 x 的点权增加 a 。
2. 把某个节点 x 为根的子树中所有点的点权都增加 a 。
3. 询问某个节点 x 到根的路径中所有点的点权和。

Algorithm Design

我们考察操作对于查询的贡献。
对于操作1，如果节点y是节点x的后代，那么可以贡献
对于操作2，如果节点y是节点x的后代，那么可以贡献
我们可以使用两个树状数组来维护贡献。
