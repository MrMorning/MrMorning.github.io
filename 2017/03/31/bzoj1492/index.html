Posted on 2017-03-31 Edited on 2020-10-08

Brief Description

某交易所发行A券和B券两种金券, 每天有两种决策:

  • 卖出金券: 顾客提供一个[0,100]的实数, 将k%的A券和k%的B券按照当天汇率兑换.
  • 买入金券, 顾客提供元人民币, 兑换价值元的金券, A券和B券的比例为.

开始有元, 求天后的最大收益.

Algorithm Design

我们不妨设为第i天的钱全部兑换, 可以获得多少A券, 可以知道, 一定有一种最优决策, 每次都是全买全卖的, 所以由此可以设计出方程:

$$f(i) = \max_{j} {f(j)A_i+\frac{f(j)}{Rate_j}*B_i} * \frac{Rate_i}{A_iRate_i+B_i}$$.

然后, 这个方程是1D/1D的, 显然可以用斜率优化:

通过打一些小的数据的表, 我们可以发现, 是单调增的, 我们就可以考虑使用这个式子斜率优化.

具体地, 我们有两种方法:

  • 平衡树: 使用splay维护这个上凸壳, 然后每次二分查找.
  • CDQ分治.

我们选择代码量相对少的CDQ分治.
